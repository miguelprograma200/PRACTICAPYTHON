Calcular si un numero es positivo o negativo

El Numero {{par_impar}}



Descripción del Ejercicio

Enunciado: Escribe un programa que pida un número y determine si es positivo, negativo o cero. Además, si el número es positivo, indique si es par o impar.

  • Entrada: Un número.
  • Proceso: Usar condicionales:
    - Si es mayor que 0, verificar si es par o impar.
    - Si es menor que 0, indica que es negativo.
    - Si es igual a 0, indica que es cero.
  • Salida: La clasificación del número como positivo, negativo o cero, y si es par o impar cuando es positivo.
